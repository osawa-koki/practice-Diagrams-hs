# practice-Diagrams-hs

HaskellでDiagramパッケージを使用して画像を描写します。

![ひまわり](./fruits/sunflower.svg)

## 実行方法

```shell
# パッケージのインストール
stack setup

# コンパイル
stack build

# 実行
stack exec -- practice-Diagrams-hs-exe <コマンドライン引数>
```
